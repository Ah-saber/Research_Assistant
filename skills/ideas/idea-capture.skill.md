# /idea-capture - 捕捉和记录Idea

## 触发词
- "记录Idea"
- "捕捉想法"
- "记下来"
- "有个想法"
- "Idea记录"

## 功能描述
快速记录研究想法和灵感到Inspiration目录，自动添加标签和关联内容。

## 存储位置

**Inspiration目录**：`C:\Note\MyNote_Obs\Inspiration\`

- 可自由访问，无需授权
- 临时存放，用户后续手动移至IDEA目录
- 按日期和主题组织

## 捕捉方式

### 1. 快速记录
用于：突发的简短想法

```
用户: "记录Idea：用注意力机制来解释CNN决策过程"
你: [自动创建Idea笔记]
```

### 2. 完整想法
用于：需要详细描述的想法

包含：
- 标题
- 详细描述
- 动机
- 可行性分析
- 相关资源

### 3. 从笔记提取
用于：从现有笔记中提取想法

```
用户: "从这篇论文笔记提取Idea"
你: [分析笔记，提取潜在想法]
```

## Idea状态流转

```

| thinking | 正在思考和研究 |

## 智能功能

### 自动标签推断
根据Idea内容自动推荐标签

### 自动链接相关论文
- 检测Idea内容中提到的论文/方法/模型
- 使用Glob搜索匹配的论文笔记文件
- 自动创建Wikilink到相关论文笔记
- 使用别名格式：`[[完整文件名|显示名]]`

**工作流程**：
1. 分析Idea内容，提取关键词（论文标题、方法名、模型名）
2. 对每个关键词使用Glob在Paper目录搜索匹配的.md文件
3. 如果找到精确匹配或高度相似的文件，添加到Frontmatter的related字段
4. 在"相关工作"部分创建带描述的Wikilink

**示例**：
```python
# 提取关键词
keywords = extract_keywords(idea_content)  # ["扩散模型", "DDPM", "生成式模型"]

# 搜索匹配文件
for keyword in keywords:
    matches = glob_search(f"**/Paper/**/*{keyword}*.md")
    if matches:
        related_papers.append(matches[0])

# 添加链接
related: [["DDPM论文|Denoising Diffusion Probabilistic Models"], ["扩散模型主题"]]
```

## 与 obsidian-markdown 的关系

**分工明确**：
- **obsidian-markdown**：提供 Obsidian 格式语法参考文档
  - Frontmatter、Wikilink、Callout、标签的语法规范
- **/idea-capture**：提供Idea捕捉工作流
  - 收集想法 → 搜索相关论文 → 创建Idea笔记

**不是重复造轮子**：
- obsidian-markdown 是**文档技能**，不执行文件操作
- 本技能是**工作流封装**，使用 Write + Glob 创建笔记和建立链接
- 两者是**参考与执行**的关系

**使用的Obsidian特性**：
- Frontmatter：Idea笔记包含完整元数据（status、priority、theme等）
- Wikilink：自动链接到相关论文笔记，使用[[完整文件名|显示名]]格式
- Callout：在"挑战与风险"部分添加提醒块
- 标签：层级标签格式，使用英文术语
